## Pong (ASCII игра на C)

**Описание:**
Учебный проект для закрепления **структурного программирования** на языке **C (C99)**. Основная цель — реализовать символьную версию классической игры **Pong** для двух игроков с пошаговым режимом, управлением через клавиатуру и подсчетом очков.

**Что было выполнено:**

* Реализована базовая игра **Pong** в пошаговом режиме (`src/pong.c`)

  * Игровое поле: 80×25 символов
  * Размер ракетки: 3 символа
  * Мяч: 1 символ
  * Управление ракетками: **A/Z** и **K/M**
  * Пропуск действия: пробел
  * Победа при 21 очке
* Бонус: реализован интерактивный режим игры в реальном времени (`src/pong_interactive.c`) с использованием дополнительных библиотек для обработки клавиш и вывода ASCII-графики

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
* Библиотеки для интерактивного режима (бонус): `<ncurses.h>` или аналогичные
