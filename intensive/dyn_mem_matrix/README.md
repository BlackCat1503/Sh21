## Динамическая память, матрицы и операции над ними на C

**Описание:**
Учебный проект для закрепления работы с динамическим выделением памяти, массивами и матрицами на языке **C (C99)**. Включает ввод/вывод массивов и матриц, сортировку, арифметику и графическое представление данных в консоли.

**Что было выполнено:**

* Динамическое выделение памяти для массивов (`sort.c`)
* Исправление утечек памяти (`sort_no_leak.c`)
* Работа с матрицами: 1 статический и 3 динамических способа выделения памяти (`matrix.c`)
* Расширение функционала матрицы: поиск максимальных элементов по строкам и минимальных по столбцам (`matrix_extended.c`)
* Воссоздание рисунка из массива (`picture.c`)
* Сортировка строк матрицы по сумме элементов (`matrix_sort.c`)
* Арифметика матриц: сложение, умножение, транспонирование (`matrix_arithmetic.c`)
* Решение финальной задачи и сохранение результата (`key10.txt`)

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Динамическая память: `<stdlib.h>` (malloc, calloc, free)
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
