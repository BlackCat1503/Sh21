## Vim, Bash-скрипты и конфликты Git

**Описание:**
Учебный проект для углубления навыков работы с **Vim**, **bash-скриптами**, **Git** и **логированием изменений** в рамках консольного квеста.

**Что было выполнено:**

* Редактирование файлов в **Vim** с внесением версионных изменений
* Ведение журнала изменений файлов с хешированием (**SHA-256**)
* Разработка bash-скрипта для автоматической замены текста в файлах (`edit.sh`)
* Разработка bash-скрипта для анализа логов (`log_analyzer.sh`)
* Обработка ошибок и краевых случаев в скриптах
* Разрешение **Git-конфликтов** после неудачного merge
* (Bonus) Создание технического **Markdown-мануала по GitLab** со скриншотами

**Стек технологий:**

* Язык: Bash / Shell
* Редактор: Vim
* Git / GitHub
* Работа с текстовыми файлами и скриптами
* ОС: Linux
