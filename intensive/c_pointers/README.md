## Указатели, ввод/вывод данных, статистика и поиск на C

**Описание:**
Учебный проект для освоения работы с **указателями, массивами, вводом/выводом данных**, базовой статистикой и поиском элементов в языке **C (C99)**. Особое внимание уделялось правильной обработке ошибок и валидации ввода.

**Что было выполнено:**

* Исправлен модуль поиска максимума и минимума (`maxmin.c`)
* Исправлен модуль возведения элементов массива в квадрат (`squaring.c`)
* Реализован модуль статистики (`stat.c`): вывод массива, максимума, минимума, математического ожидания и дисперсии
* Реализован модуль поиска (`search.c`): поиск первого числа по заданным условиям (чётное, ≥ ожидания, по правилу трёх сигм, ≠ 0)
* Во всех заданиях реализована **валидация ввода** и вывод `n/a` при ошибках

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Математические функции: `<math.h>`
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
