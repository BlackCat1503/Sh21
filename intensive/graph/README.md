## Graph Plotter (ASCII версия на C)

**Описание:**
Учебный проект для закрепления **структурного программирования** на языке **C (C99)**. Цель — реализовать визуализацию **графиков произвольных функций** в терминале с использованием ASCII-графики.

**Что было выполнено:**

* Реализована программа `graph.c`

  * Вход: строка с выражением функции `f(x)`
  * Вычисление значений выражения через **обратную польскую нотацию (RPN)**
  * Использован алгоритм **Дейкстры (Shunting-yard)** для преобразования инфиксного выражения в постфиксное
  * Поддержка операций: `+`, `-` (унарный и бинарный), `*`, `/`, `()`, `sin(x)`, `cos(x)`, `tan(x)`, `ctg(x)`, `sqrt(x)`, `ln(x)`
  * Отображение графика в терминале ASCII (знак `*`) на поле 25×80
  * Округление координат по математическим правилам
  * Центр координат: левая граница, середина по вертикали (0,12)
  * Область определения: `[0; 4π]`, область значений: `[-1; 1]`
  * Вывод только графика, без осей или других символов

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Математические функции: `<math.h>`
* Стандартная библиотека: `<stdlib.h>`, `<string.h>`
* Используются собственные структуры данных (стек для RPN, список лексем)

**Сборка:**

* Сборка через Makefile
* Стадия сборки: `all`
* Исполняемый файл: `build/graph`
