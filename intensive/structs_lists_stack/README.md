## Структуры, объединения, односвязный список и динамический стек на C

**Описание:**
Учебный проект для закрепления работы с **структурами и объединениями**, а также динамическими структурами данных на языке **C (C99)**. Включает работу с массивами структур, односвязным списком и стеком для целых чисел, модульное тестирование и управление ресурсами.

**Что было выполнено:**

* Создание структуры дверей (`door_struct.h`) и модификация модуля управления дверьми (`dmanager_module.c`)
* Сортировка массива структур дверей и установка статуса «закрыта» для всех дверей (Quest 1)
* Реализация односвязного списка дверей (`list.c` / `list.h`) с функциями инициализации, добавления, поиска, удаления и освобождения памяти
* Модульные тесты для функций add_door и remove_door (`list_test.c`) (Quest 2)
* Реализация динамического стека для целых чисел (`stack.c` / `stack.h`) с функциями init, push, pop, destroy
* Модульные тесты для push и pop стека (`stack_test.c`) (Quest 3)

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Динамическая память: `<stdlib.h>` (malloc, free)
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
