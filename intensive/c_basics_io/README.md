## Основы C: ввод, вывод и арифметика

**Описание:**
Учебный проект для освоения базовых операций языка **C (C11)**: ввода-вывода, арифметики, работы с вещественными числами, проверок ошибок и логических вычислений.

**Что было выполнено:**

* Написана программа вывода приветствия (`hello.c`)
* Реализован ввод имени и форматированный вывод (`named_hello.c`)
* Реализованы арифметические операции с проверкой ошибок (`arithmetic.c`)
* Реализован поиск максимума через отдельную функцию (`max.c`)
* Реализован расчёт сложной математической функции с использованием `math.h` (`important_function.c`)
* Исправлено сравнение чисел с плавающей точкой через **epsilon** (`float_compare.c`)
* Реализована проверка попадания точки внутрь окружности (`crack.c`)
* Во всех заданиях реализована **валидация ввода** и вывод `n/a` при ошибках

**Стек технологий:**

* Язык: C (C11)
* Стандартный ввод/вывод: <stdio.h>
* Математические функции: <math.h>
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
