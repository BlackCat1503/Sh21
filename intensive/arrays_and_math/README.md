## Основы C: сортировка, циклический сдвиг и арифметика больших чисел

**Описание:**
Учебный проект для закрепления навыков работы с массивами и арифметикой на языке **C (C11)** без использования динамической памяти и стандартной библиотеки `stdlib.h`. Включает сортировку, циклический сдвиг, работу с несколькими массивами и арифметику больших чисел.

**Что было выполнено:**

* Реализована сортировка массива из 10 целых чисел без `stdlib.h` (`sort.c`)
* Добавлены две эффективные сортировки с оценкой сложности `O(n log n)` (`fast_sort.c`)
* Создана программа обработки нескольких массивов с выделением элементов по сумме (`key9part1.c`)
* Реализован циклический сдвиг элементов массива (`cycle_shift.c`)
* Добавлена арифметика больших чисел: сложение и вычитание с проверкой переполнения (`key9part2.c`)
* Во всех заданиях реализована **валидация ввода** и вывод `n/a` при ошибках
* Применена модульная структура и передача массивов только по указателю

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: <stdio.h>
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
