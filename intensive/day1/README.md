## Linux & Git Basics

**Описание:**
Учебный проект в формате текстового квеста, направленный на освоение базовой работы с **Linux**, **bash** и **Git** через практические задания в терминале.

**Что было выполнено:**

* Клонирование репозитория и работа с ветками Git
* Запуск и отладка bash-скриптов
* Работа с файловой системой Linux (создание, удаление, редактирование файлов)
* Управление процессами (поиск и завершение процессов)
* Редактирование конфигурационных файлов через консольные редакторы
* Автоматизация действий через shell-скрипты
* Генерация ключа из набора файлов и объединение через bash-утилиты

**Стек технологий:**

* **Linux**
* **Bash**
* **Git**
* **Shell-скрипты**

**Результат:**
Все квесты успешно пройдены, конфигурация восстановлена, ключ собран, доступ открыт.
