## Game of Life (ASCII версия на C)

**Описание:**
Учебный проект для закрепления **структурного программирования** на языке **C (C99)**. Основная цель — реализовать визуализацию **The Game of Life** Джона Конвея с использованием ASCII-графики в терминале.

**Что было выполнено:**

* Реализована базовая версия игры (`src/game_of_life.c`)

  * Игровое поле: 80×25 клеток
  * Клетки живые/мертвые
  * Поле **замкнуто на себя** (тороидальная топология)
  * Настройка скорости игры в интерактивном режиме
  * Поддержка начальной инициализации через stdin
  * Подготовлено минимум 5 файлов с начальными конфигурациями (стабильные, периодические, движущиеся фигуры и т.д.)

**Стек технологий:**

* Язык: C (C99)
* Стандартный ввод/вывод: `<stdio.h>`
* Компиляция: gcc / clang
* ОС: кроссплатформенно (Linux, Windows, macOS)
